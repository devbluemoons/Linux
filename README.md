### linux self-study
  
###### [check version]
```sh
uname -a
#OR
cat /etc/issue
```
  
###### [access root authority]
```sh
su - root
#and then type password
```
  
###### [find directory or file name]
```sh
# sudo find / -name [name] 

sudo find / -name tomcat8
```
  
###### [service process]
```sh
# sudo service [process/-version] start or stop

sudo service tomcat8 start
sudo service tomcat8 stop
```
