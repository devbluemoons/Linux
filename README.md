### linux self-study
  
###### [check version]
```sh
uname -a
#OR
cat /etc/issue
```
  
###### [find directory or file name]
```sh
# sudo find / -name [name] 

sudo find / -name tomcat8
```
  
###### [service tomcat]
```sh
# sudo service [tomcat-version] start or stop

sudo service tomcat8 start
sudo service tomcat8 stop
```
